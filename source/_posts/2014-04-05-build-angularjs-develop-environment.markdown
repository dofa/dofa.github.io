## git clone "PhoneCat Tutorial App"

	git clone https://github.com/angular/angular-phonecat.git


## 包安装

在angular-phonecat目录下安装包：
	
	cd angular-phonecat

	npm install -d

## 本地浏览

开启服务

	npm start

在浏览器中输入：

	http://localhost:8000/app/index.html
