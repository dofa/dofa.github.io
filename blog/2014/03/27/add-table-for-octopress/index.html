Octopress表格设置

Mar 27th, 2014

Octopress默认的表格没有边框，比较难看。需要增加一个css。

新建一个 data-table.css，放到source/stylesheets下面

在source/_includes/head.html增加如下代码：

在需要该样式的post中增加

styles: [data-table]

即可。

表格写法

左对齐表头 | 中间对齐表头 | 右对齐表头
:----------|:------------:|----------:
左对齐数据 |中间对齐数据  |右对齐数据
第二行数据 |也是第二行    |还是第二行
懒得写了...|.....         |.....
长数据，以便看出表头的对齐|长数据，以便看出表头的对齐|长数据，以便看出表头的对齐
