Apr 3rd, 2014 | Comments

Frames用于表现HTML页面当前窗体的中的框架集合

Frames对象

  • frames对象是window对象的属性
  • 如果页面使用框架，将产生一个框架集合frames，在集合中可用数字(从0开始，从左到右，逐行索引)或名字索引框架

框架集引用中使用的对象

对象 位置
window 当前框架
top 最顶层的框架，就是浏览器窗体
parent 包含当前框架的父框架
self 当前框架，总是等于window对象
